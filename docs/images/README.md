# User Guide screenshots

Drop PNGs here with these exact filenames and they'll render in
[`../USER_GUIDE.md`](../USER_GUIDE.md) with no further edits.

| Filename | What to capture |
| --- | --- |
| `dashboard-overview.png` | Hero shot: a full new-tab dashboard — header with pinned links, a Google search, a couple of groups, and 2–3 widgets on a subtle wallpaper. |
| `edit-mode.png` | The dashboard in edit mode — Clear / Cancel / Save buttons visible in the header, plus an item mid-drag or showing its ✕ / ⋯ controls. |
| `group.png` | A colored Group folder holding several link tiles, with its header controls (title, + add, color, columns, grid/list). |
| `reminders.png` | The Reminders widget — a few reminders (one overdue/red, one recurring with the 🔁 icon) and the add row with the date/time picker and recurrence dropdown open. |
| `popup.png` | The "Save to MonoStart" toolbar popup — current-page preview, the Destination dropdown, and the Save Link button. |
| `theme-panel.png` | The Theme & Appearance modal — Light/Dark/Device toggle, primary-color swatches with the custom-hue slider, and the Background section with blur/dim sliders. |

**Tips:** capture at a consistent width (~1280px for full-dashboard shots), use a clean
demo layout, and prefer light mode for legibility (or include one of each where it matters).
